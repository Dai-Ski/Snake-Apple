# 🍎 Eve's Apple

A modern, highly-polished desktop Snake engine built from scratch in Java, fully packaged as a standalone Mac application.

![Gameplay](assets/gameplay_screenshot.png)

## 🚀 How to Play

No terminal or installation required! Simply download the **Eve's Apple.app** from this repository and double-click to play! 

*(Requires Java installed on your Mac)*

## 🎮 Controls
- **Movement:** `W` `A` `S` `D`  *or*  `Arrow Keys`
- **Pause / Play:** `Spacebar`
